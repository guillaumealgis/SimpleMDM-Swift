Protocol Gettable​Resource 

public protocol GettableResource: IdentifiableResource

A protocol describing resource types that can be fetched independently, using their unique identifier (these resources adopt the IdentifiableResource protocol).

Relationships

Conforms To

IdentifiableResource

A protocol describing resource types of which multiple instances of can exists. These resources have an identifier which is unique per instance of the resource.
