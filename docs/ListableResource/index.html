Protocol Listable​Resource 

public protocol ListableResource: GettableResource

A protocol describing resource types that we can get a list of.

Relationships

Conforms To

GettableResource

A protocol describing resource types that can be fetched independently, using their unique identifier (these resources adopt the IdentifiableResource protocol).

Types Conforming to Listable​Resource

App

An App represents an app in your app catalog. You can use AppGroup to install apps to your devices.

AppGroup

An app group is an object that pairs App resources with DeviceGroup resources for the purpose of pushing apps to devices.

CustomAttribute

Custom attributes defined in your account.

CustomConfigurationProfile

A custom configuration profile created with AppleConfiguration and applied to devices.

Device

A device registered in your SimpleMDM account.

DeviceGroup

A group of device, used to assign apps and profiles to devices.

App.ManagedConfig

A managed app configuration (a key-value pair) associated with an App.

SearchableResource

A protocol describing resource types for which we can get instances without knowing their identifier, by searching for value in their properties. Which properties are defined by the SimpleMDM implementation server-side. See the online SimpleMDM documentation.

get​All(completion:​) 

static func getAll(completion: @escaping CompletionClosure<[Self]>)

Fetch the list of all resources of this type.

Because the SimpleMDM API enforces pagination when getting a list of resources, this method will fetch the resources page by page, and concatenate the results of each page. This means that if the total count of resources fetched is high (more than a hundred of resources), calling this method may end up making multiple HTTP requests to the SimpleMDM API.

Prefer using the Cursor class over this convenience method when fetching large lists of resources (see Discussion).
