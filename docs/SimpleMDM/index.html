api​Key 

public static var apiKey: String?

Your private SimpleMDM API key.

You must set this property to an non-empty string before using any other object of this library. Failure to do so will result in most methods returning a SimpleMDMError.aPIKeyNotSet error.

Your API key can be found in your SimpleMDM account, under Settings > API > Secret Access Key.
